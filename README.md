# Java Basics 🚀

A collection of beginner-friendly Java programs covering fundamental programming concepts. This repository is perfect for students learning Java from scratch.

## 📚 Topics Covered
- Basics & Variables
- User Input (`Scanner`)
- Conditional Statements (`if-else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Pattern Programs
- Methods & Functions
- Practice Questions

## 🛠️ Tech Stack
- Java
- VS Code
